# stock_vue_frontend

这是与 `stock_web_project` 并列的独立 Vue 3 + Vite 前端工程骨架。

## 已搭建内容

- Vue 3 + Vite + TypeScript
- Vue Router
- Pinia
- 基础 App Layout
- 左侧导航 / 顶部工作区 / 右侧详情抽屉
- 7 个模块页面骨架
- API 服务层按模块拆分
- 基础全局样式

## 目录启动

```bash
cd stock_vue_frontend
npm install
npm run dev
```

默认开发代理会把 `/api` 转发到 `http://127.0.0.1:8080`。
